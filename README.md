# Portfólio - Daniel Donaire

Portfólio pessoal feito que atua como um "cartão de visitas" digital. Contém a página inicial, uma página sobre mim e uma página com os meus projetos.

### Linguagens utilizadas:
* HTML, CSS e JavaScript.

### Funcionalidade em JS:
* **Saudação Automática:** O site detecta o horário do seu computador e muda a mensagem de boas-vindas (Morning, Afternoon ou Evening).
